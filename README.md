# Internal Messaging Module for Online University

## Overview  
This project contains the Internal Messaging module developed as part of a larger Online University Management application for the Generation Italy .NET Developer Program (Febâ€“May 2025). It enables users (students, teachers, admins) to exchange messages within the platform.

## Technologies  
- ASP.NET MVC  
- C#  
- LINQ  
- Bootstrap  
- JavaScript  
- SQL Server  

## Features  
- User-to-user messaging with dynamic routing  
- Message storage in a relational database  
- Responsive UI using Bootstrap and JavaScript for enhanced interactivity  
- Integration with the main application user roles
